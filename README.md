🎯 Number Guessing Game
📌 Task 2 – Java Development Internship

A simple Java console-based game where the user tries to guess a randomly generated number.

🚀 How It Works
The system generates a random number (1–100).
The user enters a guess.
The program gives hints:
📉 Too Low
📈 Too High
✅ Correct Guess
The game continues until the correct number is guessed.
🛠️ Tech Used
Java (Core Java)
Scanner & Random classes
▶️ Run the Project
javac Main.java
java Main

🎯 Outcome
Learned Java basics, loops, and conditionals
Practiced user input handling and logic building
